# Clinic Management Web Application
Built with a <strong>ReactAdmin + Vite frontend</strong> and an <strong>Express.js + MongoDB backend</strong>, the system uses <strong>JWT-based authentication</strong> and access restrictions to ensure secure handling of sensitive clinic data.

Deployed on Render for reliable cloud hosting, the application is designed for <strong>scalability, maintainability, and efficient clinic operations</strong>.

Built RESTful APIs with Express.js and modeled clinic domain entities using Mongoose schemas for:

1. users
2. employees
3. patients
4. departments
5. appointments
6. permissions

Implemented:

1. JWT auth
2. route protection
3. RBAC permissions
4. modular API architecture
5. reusable admin resources
6. secure CRUD operations
7. cloud deployment via Render
